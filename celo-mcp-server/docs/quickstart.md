# Quick Start

## Prerequisites
- Node.js 20+
- A Celo wallet private key
- (Optional) Alfajores test CELO from faucet.celo.org

## 1. Clone and install
\\\ash
git clone https://github.com/your-org/celo-mcp-server
cd celo-mcp-server
npm install
\\\

## 2. Configure
\\\ash
cp .env.example .env
# Edit .env and add your PRIVATE_KEY
\\\

## 3. Build
\\\ash
npm run build
\\\

## 4. Add to Claude Desktop
See docs/claude-desktop-setup.md
